# StrategyTune

[StrategyTune](https://strategytune.com) FREE web app for manual backtesting of trading strategies.

## Features
- Manual backtesting with smooth price replay  
- Drag timeline like in video player. jump through time forward and backward
- Cancallable trading actions
- Precise replay speed control from 1x to 50,000x
- No login required
- Cloud storage for backtesting progress for registered users
- Tick-by-tick data quality with real spread  
- Supports Forex, Crypto, Indices, Stocks, Commodities
